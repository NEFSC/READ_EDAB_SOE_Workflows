# Bottom Trawl Survey Strata and associated EPUs

Ecological Production Units (EPUs) are defined (for Bottom Trawl Survey
data) by a set of strata in which the survey samples were taken

## Usage

``` r
epu_strata
```

## Format

A data frame

- EPU:

  Ecological Production Unit

- STRATUM:

  STRATUM that comprise the EPU
